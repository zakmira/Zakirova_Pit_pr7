# ПРАКТИЧЕСКАЯ РАБОТА №7 "ОТЛАДКА ПРОГРАММЫ РАЗЛИЧНЫМИ СПОСОБАМИ (Часть 2)"
**Цель практической работы:** произвести процедуру отладки программного обеспечения встроенными средствами среды программирования Microsoft Visual Studio.

**Выполнила:** Закирова Эльмира, 3ИСиП-423 (вариант 9 - шифр Хилла)

**Предметная область:** шифр Хилла — это метод полиграммной (многобуквенной) подстановки, разработанный американским математиком Лестером Хиллом в 1929 году. Данный метод относится к классу симметричных блочных шифров и основан на применении операций линейной алгебры, в частности — умножения матриц.
